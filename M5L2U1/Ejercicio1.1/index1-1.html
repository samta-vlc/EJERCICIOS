Calcular el volumen y el área de un prisma rectangular (ortoedro)

Si llamamos a a la longitud de un ortoedro, b a su ancho o profundidad y c a su altura, podemos definir las siguientes fórmulas:

Volumen = a · b · c

Área = 2 · (a · b + a · c + b · c)

Área base = a · b

Área lateral = 2 · c · (a + b)

Área total = 2 · Área base + Área lateral

Volumen* = Área base · c

Tenemos un ortoedro con las siguientes medidas:

  • a, longitud de 23 cm;
  • b, ancho o profundidad 16 cm, y;
  • c, altura de 13 cm.
